**Язык:** Python 3.x

 Описание

Графическое приложение для генерации случайных цитат с возможностью фильтрации по автору и теме, сохранением истории и загрузкой данных из JSON.

 Возможности

- Генерация случайной цитаты из предопределённого списка.
- Фильтрация цитат по автору и теме.
- Отображение истории сгенерированных цитат.
- Сохранение истории и базы цитат в файл `quotes.json`.
- Проверка на пустые строки при добавлении новых цитат (базовые цитаты уже заполнены).
- Использование Git для контроля версий.

 Запуск

1. Установите Python 3.x.
2. Установите зависимости (если нужны): `pip install -r requirements.txt` *(в данном проекте только стандартная библиотека)*.
3. Запустите: `python quotes_generator.py`.

Использование

1. Нажмите кнопку «Сгенерировать цитату» для получения случайной цитаты.
2. Используйте фильтры для поиска по автору или теме.
3. История отображается в списке внизу окна.
4. Все данные сохраняются в `quotes.json`.

 Примеры использования

- Генерация: Кнопка «Сгенерировать цитату».
- Фильтр: Введите часть имени автора или выберите тему.
- История: Просмотр ранее сгенерированных цитат.
